操作系统实例：Windows

Windows 文件系统

Windows中常见的文件系统有 FAT 和 NTFS 两种

FAT文件系统

一种较早期的文件系统，因该系统采用文件分配表（File Allocation Table, FAT）来管理文件，故称为FAT文件系统。

FAT是用来记录文件所在位置的表格，其中包含的是目录项，记录了每一个文件（或目录）的属性数据。

FAT文件系统使用“簇”作为最小的数据单元。“簇”是一组连续磁盘扇区的组合，所有文件（或目录）都存放在簇中。一个文件总是占用若干个整簇。

NTFS文件系统

NTFS（New Technology File System）是目前最常用的一种Windows标准文件系统。

NTFS大幅度地改善了原有FAT文件系统的性能，提供长文件名、数据保护和恢复，能通过目录和文件许可实现安全性，并支持跨越分区。

NTFS分区结构比FAT要复杂。NTFS分区主要由引导扇区、 主文件表、系统文件和文件存储区域等4个部分组成。

操作系统的概念

操作系统的概念

操作系统，英文为Operating System，简称OS

操作系统位于计算机用户与计算机硬件之间，是管理计算机硬件的软件。它就像是一个管理硬件设备的管家，让这些硬件能够有效地协同工作，并为运行在上面的应用程序提供支持。

  • 可以将操作系统看成是计算机用户和计算机硬件之间的中介，它提供一个承上启下的作用。其目的是提供环境，使用户能够快捷、高效地使用计算机。
  • 不只是计算机上需要操作系统，几乎所有的智能设备上都可以看到操作系统的身影。我们平时使用的各种智能设备如手机、平板电脑，甚至一些电视、冰箱、洗衣机等智能家电，内部都包含了或复杂、或简单的操作系统。

操作系统的主要任务

  • 处理如管理与配置内存、决定系统资源供需的优先次序、控制输入设备与输出设备、操作网络与管理文件系统等基本事务。
  • 提供一个让用户与系统交互的操作界面。

不同视角看待操作系统

  • 计算机用户关注的是操作系统为其提供的各项服务
  • 程序开发人员关注的是操作系统为其提供的编程接口
  • 操作系统的设计人员则关注系统中各式各样模块和单元的设计及它们之间的联系

操作系统与硬件

操作系统的主要功能是管理计算机硬件（Computer Hardware）

从系统架构的角度看，整个计算机系统可以粗分为四个层次，从下至上分别是：

  1. 硬件（Hardware）
  2. 操作系统
  3. 应用程序（Application Program）
  4. 用户（User）

计算机系统的组成

冯·诺依曼结构计算机，主要有五大部件组成：

  • 运算器（Arithmetic Unit）
  • 控制器（Controller）
  • 存储器（Memory）
  • 输入设备（Input Device）
  • 输出设备（Output Device）

计算机的启动过程

当计算机开机或重启时，会运行保存在只读内存ROM中的一个叫做“引导程序”的初始程序。

引导程序先初始化系统的各个组件，从CPU寄存器、设备控制器到内存。

接着引导程序会定位操作系统内核并将其加载到内存中。

一旦操作系统内核加载到内存并运行，它就开始为系统和用户提供服务。

除了内核外，一些系统程序也在启动时加载到内存中运行并提供一些特定的服务，直到关机。

操作系统的功能和用户界面

信息系统安全与信息社会责任

信息系统安全与信息社会责任

基本概念

通用计算机系统概述

计算机安全包括两个方面的内容：物理安全和逻辑安全

  • 物理安全指系统设备及相关设施的安全性
  • 逻辑安全是指信息的完整性、保密性和可用性，即信息的安全

物理损坏，以及人为的误操作而造成的损坏，可以通过硬件的双工，或数据冗余，或有效的备份措施进行防治

信息安全的基本术语

  • 计算机信息系统
  • 可信计算机
  • 客体和主体

计算机信息的逻辑安全

  • 识别（身份鉴别）
  • 授权（访问控制）
  • 安全审核（审计）
  • 加密和签名
  • 病毒防范

计算机信息系统安全保护等级划分

  1. 用户自主保护级
  2. 系统审计保护级
  3. 安全标记保护级
  4. 结构化保护级
  5. 访问验证保护级

常用的安全技术

加密技术

加密技术是通信双方按约定的法则进行信息特殊变换的一种保密技术。

  • 将明文（Plaintext）为密文（Ciphertext）的过程称为加密（Encryption）
  • 由密文恢复出原明文的过程，称为解密（Decryption）

加密技术是现代信息系统安全的基石。加密技术包括两个重要的组成部分：算法和秘钥。

  • 算法是将普通的文本或者是可以理解的信息，与一串数字（秘钥）相结合，产生不可理解的密文的步骤
  • 秘钥是用来对数据进行编码和解码的一种参数

对称式加密算法

收发双方使用相同秘钥的密码的加密算法被称为对称式加密

对称加密算法的优点是加密速度快、保密度高

非对称式加密算法

通信收发双方使用不同秘钥的密码，称为非对称式加密

非对称式加密算法的基本思想是：甲将公钥发送给乙，乙用公钥加密数据后，将加密后的数据再发送给甲，其他人即使截获了加密数据，因为没有密钥，也无法获取加密数据的真实内容

常见的加密算法

  • DES
  • AES
  • RSA
  • Hash
  • MD5
  • SHA-1

身份认证

身份验证是指通过一定的手段，完成对用户身份的确认。

大致可以分为以下三种:

  • 基于共享密钥的身份验证
  • 基于生物学特征的身份验证
  • 基于公开密钥加密算法的身份验证

数字签名

数字签名（又称公钥数字签名）是只有信息的发送者才能产生的、别人无法伪造的一段数字串，这段数字串同时也是对信息的发送者发送信息真实性的一个有效证明

  • 数字签名是一种类似写在纸上的普通的物理签名，但是使用了公钥加密领域的技术来实现的，用于鉴别数字信息的方法
  • 一套数字签名通常定义两种互补的运算，一个用于签名，另一个用于验证
  • 数字签名是非对称密钥加密技术与数字摘要技术的应用

访问控制技术

访问控制（Access Control）指系统对用户身份及其所属的预先定义的策略组限制其使用数据资源能力的手段。通常用于系统管理员控制用户对服务器、目录、文件等网络资源的访问

  • 访问控制是系统保密性、完整性、可用性和合法使用性的重要基础，是网络安全防范和资源保护的关键策略之一，也是主体依据某些控制策略或权限对客体本身或其资源进行的不同授权访问
  • 访问控制的主要目的是限制访问主体对客体的访问，从而保障数据资源在合法范围内得以有效使用和管理
  • 为了达到上述目的，访问控制需要完成两个任务：识别和确认访问系统的用户、决定该用户可以对某一系统资源进行何种类型的访问
  • 访问控制包括三个要素：主体、客体和控制策略
  • 访问控制首先需要对用户身份的合法性进行验证，同时利用控制策略进行选用和管理工作
  • 当用户身份和访问权限验证之后，还需要对越权操作进行监控
  • 访问控制的内容包括认证、控制策略实现和安全审计
  • 访问控制的主要功能包括：保证合法用户访问受权保护的网络资源，防止非法的主体进入受保护的网络资源，或防止合法用户对受保护的网络资源进行非授权的访问

防火墙技术

防火墙是一个由计算机硬件和软件组成的系统，部署于网络边界，是内部网络和外部网络之间的连接桥梁，同时对进出网络边界的数据进行保护，防止恶意入侵、恶意代码的传播等，保障内部网络数据的安全

防火墙技术是建立在网络技术和信息安全技术基础上的应用性安全技术，几乎所有的企业内部网络与外部网络（如因特网）相连接的边界设都会放置防火墙，防火墙能够起到安全过滤和安全隔离外网攻击、入侵等有害的网络安全信息和行为

防火墙按照不同的使用场景主要可以分成四类：

  • 包过滤防火墙
  • 应用网关防火墙
  • 服务防火墙
  • 监控防火墙

病毒防护

计算机病毒的定义

编制者在计算机程序中插入的破坏计算机功能或者数据的代码，能影响计算机使用，能自我复制的一组计算机指令或者程序代码

计算机病毒是人为制造的，有破坏性，又有传染性和潜伏性的，对计算机信息或系统起破坏作用的程序

计算机病毒具有

  • 传播性
  • 隐蔽性
  • 感染性
  • 潜伏性
  • 可激发性
  • 表现性
  • 破坏性

计算机病毒的生命周期

  1. 开发期
  2. 传染期
  3. 潜伏期
  4. 发作期
  5. 发现期
  6. 消化期
  7. 消亡期

计算机病毒的分类

按照依附的媒体类型分类
  • 网络型病毒
  • 文件型病毒
  • 引导型病毒
按照计算机特定算法分类
  • 附带型病毒
  • 蠕虫病毒
  • 可变病毒
按照链接方式分类
  • 源码型病毒
  • 嵌入型病毒
  • 操作系统型病毒
按照计算机病毒攻击的系统分类
  • 攻击DOS系统病毒
  • 攻击 Windows系统病毒
  • 攻击UNIX系统的病毒

计算机病毒的传播途径

  • 通过移动存储设备进行病毒传播
  • 通过网络来传播
  • 利用计算机系统和应用软件的弱点传播

计算机病毒的特征

  • 隐蔽性
  • 破坏性
  • 传染性
  • 寄生性
  • 可执行性
  • 可触发性
  • 攻击的主动性
  • 病毒的针对性

中毒计算机的主要症状

  • 工作会很不正常
  • 莫名其妙的死机
  • 突然重新启动或无法启动
  • 程序不能运行
  • 磁盘坏簇莫名其妙地增多
  • 磁盘空间变小
  • 系统启动变慢
  • 数据和程序丢失
  • 出现异常的声音、 音乐或出现一些无意义的画面问候语等显示
  • 正常的外设使用异常，如打印出现问题，键盘输入的字符与屏幕显示不一致等
  • 异常要求用户输入口令

常见计算机病毒的示例

  • “爱虫”病毒
  • CIH病毒
  • Happy 99螟虫
  • 木马病毒、黑客病毒
  • 脚本病毒
  • 宏病毒
  • 后门病毒

计算机病毒的防范措施

  • 安装最新的杀毒软件，每天升级杀毒软件病毒库，定时对计算机进行病毒查杀，上网时要开启杀毒软件的全部监控
  • 不要执行从网络下载后未经杀毒处理的软件等；不要随便浏览或登录陌生的网站，加强自我保护
  • 培养自觉的信息安全意识，在使用移动存储设备时，尽可能不要共享这些设备
  • 用Windows Update功能打全系统补丁，同时，将应用软件升级到最新版本，避免病毒从网页木马的方式入侵到系统或者通过其他应用软件漏洞来进行病毒的传播；将受到病毒侵害的计算机进行尽快隔离；在使用计算机的过程，若发现计算机上存在有病毒或者是计算机异常时，应该及时中断网络；当发现计算机网络一直中断或者网络异常时，立即切断网络，以免病毒在网络中传播

信息系统的组成

通用计算机系统

通用计算机系统概述

现代信息系统的核心是计算机系统。自20世纪40年代诞生以来，计算技术保持高速发展，对人类社会产生了深刻广泛的影响，从而使得人类进入了信息社会。

现代通用计算机是基于冯·诺依曼结构的设计思想，称之为“冯·诺依曼”体系结构计算机系统。

计算机系统的两大组成部分

  • 计算机硬件是构成计算机系统各功能部件的集合，是由电子、机械和光电元件组成的各种计算机部件和设备的总称，是计算机完成各项工作的物质基础。

  • 计算机软件是指与计算机系统操作有关的各种程序以及任何与之相关的文档和数据的集合。计算机程序是用程序设计语言描述的、适合计算机执行的语句指令序列。

“冯·诺依曼”体系结构

  • 计算机硬件由五个基本部分组成：运算器、控制器、存储器、输入设备和输出设备。
  • 计算机内部采用二进制来表示程序和数据。
  • 采用“存储程序”的方式，将程序和数据放入同一个存储器中（内存储器），计算机能够自动高速地从存储器中取出指令加以执行。

集成电路技术和摩尔定律

  • 集成电路技术是制造微型机、小型机、大型机和巨型机的CPU的基本技术。
  • 在1965年，芯片巨人英特尔公司的创始人戈登 · 摩尔提出了著名的摩尔定律：芯片上的晶体管数量每隔18个月左右就会翻一番。

计算机的主要功能

  • 把需要的程序和数据送至计算机中。
  • 必须具有长期记忆程序、数据、中间结果及最终运算结果的能力。
  • 能够完成各种算术运算、逻辑运算和数据传送等数据加工处理的能力。
  • 能够根据需要控制程序走向，并能根据指令控制机器的各部件协调操作。
  • 能够按照要求将处理结果输出给用户。

计算机工作流程

  1. 取指令
  2. 指令译码
  3. 执行指令
  4. 访存取数
  5. 结果写回

计算机工作原理

首先读入程序和数据到主内存中。在控制器输入命令的控制下，通过输入设备把表示计算步骤的程序和计算中需要的原始数据读入到计算机的主内存中。

然后执行程序。在取指令作用下把程序指令逐条送入控制器。控制器对指令进行译码，并根据指令的操作要求向存储器和运算器发出存储、取数命令和运算命令，经过运算器计算并把结果存放在存储器内。

最后是输出结果。在控制器的取数和输出命令作用下，通过输出设备输出计算结果。

计算机系统硬件

五个基本部分

  • 运算器
  • 控制器
  • 存储器
  • 输入设备
  • 输出设备

CPU

通常将运算器和控制器统称为中央处理器，即CPU（Central Processing Unit），它是整个计算机的核心部件，是计算机的“大脑”。CPU控制计算机的运算、处理、输入和输出等工作。

  • CPU主要由运算器和控制器两大部分组成
  • 集成电路技术是制造微型机、小型机、大型机和巨型机的CPU的基本技术。

CPU的主要功能

  • 指令控制：指令控制也称为程序的顺序控制，控制程序严格按照规定的顺序执行。
  • 操作控制：为取出的指令产生一系列的控制信号(微指令)，分别送往相应的部件，从而控制这些部件按指令的要求进行工作。
  • 时间控制：有些控制信号在时间上有严格的先后顺序，例如读取存储器的数据，只有当地址线信号稳定以后，才能通过数据线将所需的数据读出（否则读出的数据是不正确的数据），从而确保计算机有条不紊地工作。
  • 数据加工：所谓数据加工，就是对数据进行算术运算和逻辑运算处理。

控制器（CU）

控制器CU（Controller Unit）由程序计数器、指令寄存器、指令译码器、时序产生器和操作控制器组成。

  • 控制器是计算机的指挥系统，完成计算机的指挥工作。
  • 控制器的基本功能是从内存读取指令和执行指令。

运算器（ALU）

运算器也称为算术逻辑单元ALU（Arithmetic Logic Unit）。

  • 运算器由算术逻辑单元、累加寄存器、数据缓冲寄存器和状态条件寄存器组成。
  • 运算器是数据加工处理部件，完成计算机的各种算术运算和逻辑运算。

CPU的架构

CPU按指令集可以主要分为如下两种类别：CISC（Complex Instruction Set Computers，复杂指令集计算机）和RISC（Reduced Instruction Set Computers，精简指令集计算机）。

  • CISC以Intel和AMD的X86 CPU为代表，而RISC以ARM和IBM Power为代表。
  • 在计算机发展的数十年历史中，产生了各种架构的CPU。目前比较流行的架构主要包括以下4种：X86、ARM、RISC-V和MIPS。其它指令架构相对小众，只在专业的领域使用。

CPU的主要参数

CPU的主要参数有主频、外频、前端总线频率、倍频、高速缓存容量、所采用的制造工艺、功耗，以及CPU所采用的处理器架构、一系列的处理器计算技术等。

存储设备概述

存储器是计算机的记忆装置，它的主要功能是存放程序和数据。

根据存储器与CPU联系的密切程度，可以将存储器分为内存储器（主存储器，简称为内存、主存）和外存储器（辅助存储器，简称为外存、附存）两大类。

  • 内存在计算机主机内，它直接与运算器、控制器交换信 息，容量虽小，但存取速度快，一般只存放那些正在运 行的程序和待处理的数据。
  • 外存存取速度慢，但存储容量大，可以长时间地保存大 量信息。
  • CPU可以直接访问内存，但不能直接访问外存

存储系统的层次结构

寄存器

  • 寄存器集成在CPU内部，是属于L0层的存储设备。

  • 寄存器一般和CPU同时钟频率，所以速度非常快。

  • 受限于工艺和成本，所以在CPU中只有少量的寄存器。而每个寄存器的大小都是8~512个字节。

高速缓存（Cache）

目前CPU中一般集成了2到3级的Cache，容量从128K到4M。

对于CPU中的Cache来说，它们也是和CPU同频率的，所以理论上执行速度和寄存器应该是相同的，但是Cache往往用来存储一些指令和数据，这样就存在一个命中的问题。当没有命中的时候，需要向下一级的存储器获取新的数据，这时Cache会被锁定（lock），所以导致实际的执行速度要比寄存器慢。

主内存

主内存是RAM随机访问存储，属于L3层的存储设备。

主内存简称为主存，即我们常说的内存。

主内存通常使用DRAM来实现。主流的内存是DDR2、DDR3、DDR4，数字越大，表示其带宽越大。

本地磁盘存储

本地磁盘存储就是通常所见的硬盘，本地磁盘存储属于L4层的存储设备，其优点是容量大、缺点是速度慢。

本地存储设备包括常见的机械硬盘和固态硬盘，以及不常见的软盘、光盘、磁带等。

  • 机械硬盘的优点是容量大，缺点是速度较慢。

​ 相对于CPU内部存储的电子结构，磁盘存储是一种机械结构。数据都通过电磁流来改变极性的方式被电磁流写到磁盘上，而通过相反的方式读回。

  • 固态硬盘的优点是速度快，缺点是容量不大，价格较贵。

​ SSD一般采用Flash Memory，速度较快。

移动存储和远程存储

移动存储指便携式的数据存储设备，现代的移动存储主要有移动硬盘、USB盘和各种记忆卡。

云存储是一种网上在线存储（英语：Cloud storage）的模式，即把数据存放在通常由第三方托管的多台虚拟服务器，而非专属的服务器上。

存储设备的主要性能指标

容量

程序和数据在计算机中以二进制的形式存放于存储器中。存储容量的大小以字节为单位来度量。常用的存储单位包括如下几种：bit、byte、word等

内存频率

内存频率一般包括了核心频率、总线频率和传输频率。核心频率是脉冲震荡信号的频率；总线频率是指内存总线的工作频率；传输频率是指实际传输数据的频率。例如常见的DDR3 1600指的是内存的传输频率。

RAM vs ROM

半导体存储器可以分成两大类：断电后数据会丢失的易失性（Volatile）存储器和断电后数据不会丢失的非易失性（Non-volatile）存储器。

  • RAM（Random-Access Memory）属于可随机读写的易失性存储器
  • ROM（Read-Only Memory）属于非易失性存储器。

输出和输出设备

输入设备和输出设备是指可以与计算机进行数据传输的硬件。

I/O

I/O（Input/Output），即输入/输出，是计算机与外部世界之间的通信。通常指数据在内部存储器和外部存储器或其他周边设备之间的输入和输出。

输入设备

输入设备是从计算机外部向计算机内部传送信息的装置，其功能是将数据、程序及其他信息，从人们熟悉的形式转换为计算机能够识别和处理的形式，然后输入到计算机的内部。

输出设备

输出设备是将计算机的处理结果传送到计算机外部，从而供计算机用户使用的装置。输出设备的功能是将计算机内部二进制形式的数据信息转换成人们所需要的或其他设备能接受和识别的信息形式。

设备控制器

I/O设备一般由机械部件和电子部件两部分组成。电子设备一般称为设备控制器。

驱动程序

对设备控制器进行控制的软件，称为驱动程序。

一般来说驱动程序由硬件设备厂商提供。

总线（Bus）

总线是各种功能部件之间传送信息的公共通信干线。

总线是由导线组成的传输线束。

数据总线（Data Bus）

数据总线既可以把CPU的数据传送到存储器或输入输出接口等其它部件，也可以将其它部件的数据传送到CPU。

数据总线的位数是微型计算机的一个重要指标，通常与微处理的字长相一致。我们说的32位计算机、64位计算机指的就是数据总线。

地址总线（Address Bus）

地址总线是专门用来传送地址的，由于地址只能从CPU传向外部存储器或I/O端口，所以地址总线总是单向三态的，这与数据总线不同。

地址总线的位数决定了CPU可直接寻址的内存空间大小。

控制总线（Control Bus）

控制总线主要用来传送控制信号和时序信号。

控制总线的传送方向由具体控制信号而定，一般是双向的，控制总线的位数要根据系统的实际控制需要而定。

数据总线和控制总线可以共用。

内部总线

在CPU内部，用于在寄存器之间和算术逻辑部件ALU以及控制部件之间传输数据所用的总线称为内部总线。

外部总线

通常所说的总线指外部总线，是CPU与内存RAM、ROM和输入/输出设备接口之间进行通讯的通路，也称系统总线。

现代计算机总线

  • 前端总线
  • 内存总线
  • 显卡总线
  • I/O总线
  • 扩展总线

USB

USB，是英文Universal Serial Bus（通用串行总线）的缩写，是一个外部总线标准，广泛用于规范电脑与外部设备的连接和通讯。

计算机系统软件

软件的定义

国际标准化组织（ISO）将软件定义为：电子计算机程序及运用数据处理系统所必需的手续、规则和文件的总称。

系统软件

系统软件一般是指控制和协调计算机及外部设备、支持应用软件开发和运行的软件

操作系统OS（Operating System）

最主要的系统软件

  • 桌面/服务器计算机的常用的操作系统包括：Linux、Windows 10、Mac OS，以及各种版本的Unix等。

  • 移动端设备常用的操作系统包括Android、IPhone IOS、鸿蒙等。

语言处理程序（汇编和编译程序等）

程序设计语言按其发展演变过程可以分为三种：

  1. 机器语言
  2. 汇编语言
  3. 高级语言

机器语言和汇编语言统称为 低级语言。

语言处理程序可以分为三种类型：

  • 汇编程序
  • 编译程序
  • 解释程序

常用的高级计算机语言：

  • C
  • C++
  • Java
  • C#
  • Python
  • Fortran

服务性程序（支撑软件）

为了帮助用户使用与维护计算机，提供服务性手段，支持其他软件开发而编制的一类程序

常用的服务型软件：

  • 工具软件
  • 编辑程序
  • 调试程序
  • 诊断程序

数据库管理系统

一种操纵和管理数据库的大型软件，用于建立、使用和维护数据库。数据库管理系统对数据库进行 统一的管理和控制，以保证数据库的安全性和完整性。

主要功能：

  • 数据定义
  • 数据操作
  • 数据库的运行管理

流行的数据库管理系统：

  • Oracle数据库
  • MySQL
  • SQL Server
  • DB2
  • Access

应用软件

在计算机各个应用领域中，为解决各类实际问题而编制的程序，其目的是帮助人们完成在特定领域中的各种工作。

常用的应用软件

  • 文字处理程序
  • 电子表格处理软件
  • 辅助设计软件
  • 实时控制软件
  • 用户应用程序

信息系统与数字社会概述

信息技术概述

信息技术发展

信息技术是在信息的获取、整理、加工、存储、传递和利用过程中采用的技术和方法。

三个不同的发展阶段

  1. 古代信息技术
    古代信息技术的特征是以文字记录为主要的信息存储手段，以书信传递为主要的信息传输方法，信息的采集、记录、传输都是在人工条件下实施的。

    三次变革：

    • 语言的产生
    • 文字的发明
    • 印刷术的发明

  2. 近代信息技术
    近代信息技术是电信革命的发展，这是信息技术的第四次变革，是以电为主角的信息技术革命。

    序幕：莫尔斯成发明了世界上第一台有线电报机

  3. 现代信息技术
    20世纪40年代,电子计算机诞生是第五次信息技术变革，标志着人类社会进入了现代信息技术发展阶段。

五次变革

  1. 语言的产生
  2. 文字的发明
  3. 印刷术的发明
  4. 电信革命
  5. 计算机技术的发明

信息技术的发展趋势

  • 信息技术的纵深化和融合化
  • 网络互联的泛在化和移动化
  • 信息处理的集中化和大数据化
  • 信息服务的智能化和个性化

信息社会三大资源

  • 信息

  • 手机支付
  • 智能导航
  • 在线商城
  • 在线社交

信息社会的内涵

  • 以人为本、包容性和全面发展的社会
  • 可持续发展的社会
  • 信息和知识是信息社会的重要资源

信息社会的主要特征

  • 信息经济
  • 网络社会

  • 在线政府

信息系统及其应用

信息系统的组成

信息系统由五个主要组成部分

硬件

信息系统中的硬件包括计算机硬件及其他设备

  • 计算机硬件主要由运算器、控制器、存储器、输入设备和输出设备组成。
  • 移动终端设备（如手机、平板电脑）、通信网络设备（如交换机、路由器），数据采集设备（如传感器）等。

软件

信息系统的软件是指用以支撑信息系统运行的所有程序和文档的总和。

  • 系统软件是管理、控制和维护信息系统的软件，包括操作系统、各种计算机语言处理软件（程序设计语言及其编译、解释程序，调试、查错程序等）、数据库管理软件等。
  • 应用软件是为了完成某种具体的应用或用途而开发的软件，比如办公软件、即时通信软件、企业管理软件、游戏软件等。

网络

信息系统中的网络是指将分布在不同地理位置中的多个物理设备进行连接，实现计算机与计算机、人与计算机、人与人之间的资源共享和通信。

  • 网络包括有线网络（如光纤网）和无线网络（如移动通信、卫星通信）。

数据

数据是信息系统中的重要资源，包括数值、文本、图像、声音、视频、动画等多种形式。

  • 信息是加工处理后的数据。
  • 人们设计和开发信息系统的目的就是为了采集数据、管理数据、分析数据，从数据中挖掘出更多有用的信息为人们提供服务。
  • 信息系统处理的目标对象就是数据，数据的管理与分析是绝大多数信息系统所具备的功能。

用户

信息系统的用户范围非常广泛，凡是和信息系统相关的人员都是信息系统的用户，包括信息系统的设计者、开发者、使用者、维护者和管理者。

  • 所有用户都应该自觉遵守信息社会中的道德准则和法律法规，负责任地发布、使用与传播信息，合理地使用信息系统。
  • 用户也应该了解信息系统中可能存在的安全风险，知道常用的信息安全风险防范方法，养成规范的信息系统操作习惯,树立信息安全意识。

信息系统的功能

信息系统的应用

  1. 输入
    通过输入设备或数据采集设备将系统所需要的数据收集并记录下来，转换成信息系统要求的规范格式
  2. 存储
    利用存储设备和相关软件对数据进行存储和管理
  3. 处理
    对系统中的数据进行处理和加工，将数据转换为有用的信息
  4. 输出
    通过输出设备输出经过信息处理生成的有用信息
  5. 控制
    对构成系统的各种信息处理设备进行控制和管理

  • 人们成为“数字公民”
  • 智能服务平台层出不穷
  • 数据成为了驱动社会发展的核心生产要素，同时，也需要全新的劳动者素养

数字社会的特征

  • 跨域连接与全时共在
  • 跨域连接与全时共在
  • 行动自主和持续互动
  • 高效协作和智能操控
