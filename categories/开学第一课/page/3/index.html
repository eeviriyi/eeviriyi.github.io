信息系统与数字社会概述

信息系统与数字社会概述 信息技术概述 信息技术发展 信息技术是在信息的获取、整理、加工、存储、传递和利用过程中采用的技术和方法。 三个不同的发展阶段 古代信息技术 古代信息技术的特征是以文字记录为主要的信息存储手段，以书信传递为主要的信息传输方法，信息的采集、记录、传输都是在人工条件下实施的。 三次变革： 语言的产生 文字的发明 印刷术的发明 近代信息技术 近代信息技术是电信革命的发展，这是信息技术的第四次变革，是以电为主角的信息技术革命。 序幕：莫尔斯成发明了世界上第一台有线电报机 现代信息技术 20世纪40年代,电子计算机诞生是第五次信息技术变革，标志着人类社会进入了现代信息技术发展阶段。 五次变革 语言的产生 文字的发明 印刷术的发明 电信革命 计算机技术的发明 信息技术的发展趋势 信息技术的纵深化和融合化 网络互联的泛在化和移动化 信息处理的集中化和大数据化 信息服务的智能化和个性化 信息社会及其特征 信息社会三大资源 信息 物质 能源 信息社会场景 手机支付 智能导航 在线商城 在线社交 信息社会的内涵 以人为本、包容性和全面发展的社会 可持续发展的社会 信息和知识是信息社会的重要资源 信息社会的主要特征 信息经济 网络社会 数字生活 在线政府 信息系统及其应用 信息系统的组成 信息系统由五个主要组成部分 硬件 信息系统中的硬件包括计算机硬件及其他设备 计算机硬件主要由运算器、控制器、存储器、输入设备和输出设备组成。 移动终端设备（如手机、平板电脑）、通信网络设备（如交换机、路由器），数据采集设备（如传感器）等。 软件 信息系统的软件是指用以支撑信息系统运行的所有程序和文档的总和。 系统软件是管理、控制和维护信息系统的软件，包括操作系统、各种计算机语言处理软件（程序设计语言及其编译、解释程序，调试、查错程序等）、数据库管理软件等。 应用软件是为了完成某种具体的应用或用途而开发的软件，比如办公软件、即时通信软件、企业管理软件、游戏软件等。 网络 信息系统中的网络是指将分布在不同地理位置中的多个物理设备进行连接，实现计算机与计算机、人与计算机、人与人之间的资源共享和通信。 网络包括有线网络（如光纤网）和无线网络（如移动通信、卫星通信）。 数据 数据是信息系统中的重要资源，包括数值、文本、图像、声音、视频、动画等多种形式。 信息是加工处理后的数据。 人们设计和开发信息系统的目的就是为了采集数据、管理数据、分析数据，从数据中挖掘出更多有用的信息为人们提供服务。 信息系统处理的目标对象就是数据，数据的管理与分析是绝大多数信息系统所具备的功能。 用户 信息系统的用户范围非常广泛，凡是和信息系统相关的人员都是信息系统的用户，包括信息系统的设计者、开发者、使用者、维护者和管理者。 所有用户都应该自觉遵守信息社会中的道德准则和法律法规，负责任地发布、使用与传播信息，合理地使用信息系统。 用户也应该了解信息系统中可能存在的安全风险，知道常用的信息安全风险防范方法，养成规范的信息系统操作习惯,树立信息安全意识。 信息系统的功能 信息系统的应用 输入 通过输入设备或数据采集设备将系统所需要的数据收集并记录下来，转换成信息系统要求的规范格式 存储 利用存储设备和相关软件对数据进行存储和管理 处理 对系统中的数据进行处理和加工，将数据转换为有用的信息 输出 通过输出设备输出经过信息处理生成的有用信息 控制 对构成系统的各种信息处理设备进行控制和管理 数字社会及其发展 什么是数字社会 数字化、网络化、大数据、人工智能等现代信息技术的快速发展和广泛应用，孕育的一种新的社会文化形态。...

数据的采集与预处理

数据的采集与预处理 数据的采集 人工获取 通过调查或者访谈的方法获得数据。问卷是调查、访谈时常用的数据采集工具，既可以采用纸质问卷，也可以制作并发布网络调查问卷。 观察自然现象或者进行科学实验，并记录通过观察或者实验得到的数据。 查找文献资料获取数据。文献资料既包括正式出版、发行的纸质书刊、报表、年鉴，也包括政府机构、职能部门网站发布的公报、统计信息、研究报告等，还包括企业、机构网站上提供的数据库数据。 自动采集 物联感知采集：对于在物联网中使用电子标签或无线终端标识的智能化物体，通过传感器感知它们的数据变化，采集相关数据，并利用通信技术上传至网络信息中心进行存储。 视频监控采集：借助不同监控点的摄像机采集监控区域的数据。 网络平台采集：主要是指用户在访问网站或使用应用程序时，网站服务器上安装的采集软件自动采集用户的各种行为数据。 几种常见的数据采集方法 搜索引擎是一种通过网络平台获取数据的方式。搜索引擎其实是一种叫做SPIDER，也就是蜘蛛爬虫的程序。互联网的网页信息通过超级链接相互连接，像一张大型蜘蛛网。搜索引擎上通过超级链接爬取能够访问的所有网页信息，然后将这些网页信息进行关键词的识别、缓存，最后生成自己的索引，存到索引库。 爬虫也是通过网络获取数据的方法之一。根据需求编写爬虫程序可以更灵活准确、方便快捷地获取各种数据。但是要注意并不是所有的数据都可以随意爬取和使用。《中国人民共和国网络安全法》及最新刑事司法解释规定“非法提供”和“非法获取”公民个人信息构成犯罪的都可以入刑。因此在使用爬虫获取数据时，必须要遵守相关的法律法规。 网络调查问卷也是通过网络获取数据的一种方式。当需要获取的数据网络上不存在的时候，可以通过发放网络调查问卷的形式获取数据。在线调查问卷有很多现成的免费的工具可以使用。 数据的预处理 由于采集到的数据可能有缺失、重复或错误，而后面用于数据分析和数据可视化的数据必须保证数据的质量，即保证数据的完整性、统一性和准确性，因而需要进行必要的整理。 数据预处理是开展数据处理和分析前的重要准备工作，指通过去重、补漏和勘误等方法，删除重复数据、补全缺失数据和校正错误数据，并对数据进行统一和标准化处理，使采集到的数据符合数据分析的需求。 数据处理 数据的输入或导入 数据一般来源于两个方面：手动输入和外部导入。 数据的输入 数值型数据 文本型数据 日期与时间型数据 逻辑型数据 数据的导入 文本文件的导入 数据的计算 公式与函数 公式是从等号开始的一个表达式，由数据、运算符、函数、单元格或区域引用等组成。 数值数据 文本数据 日期/时间数据 逻辑数据 数据的排序 简单排序 复杂排序 自定义排序 数据的筛选 自动筛选 高级筛选 数据的分类汇总

数据与信息

数据与信息 数据、信息与知识 数据 是对客观事物属性的描述，是进行记录并可以鉴别的符号 信息 信息是对数据进行加工处理的结果，是有意义的数据。 信息的特征 普遍性 传递性 共享性 依附性 可处理性 时效性 真伪性 价值相对性 数据、信息与知识的关系 知识是对信息进一步处理和提炼后形成的，是一类更高级、更抽象，而且有普遍适应性的信息。 数据、信息和知识可以看作是对客观事物感知的三个不同阶段，前者是后者的基础和前提，后者是前者的抽象和升华。从数据到信息的转化，主要理解数据间的相互关系；从信息到知识的转化，主要理解信息间的相互联系，发现和抽象其规律。 数字化与编码 认识编码 编码是指用预先规定的方法将文字、数字或其它对象转换为规定的符号组合，或将信息、数据转换成规定的电脉冲信号。简单说，编码就是信息从一种形式或格式转换为另一种形式或格式的过程。编码一般都具有特定的格式，有一定的含义，并且是对象的唯一标识。 位计数制与数制转换 进位计数制 简称进位制，是一种按进位方式实现计数的方法。 进位制一般包含基数、数码和位权三个要素。基数是数制所使用的数码的个数。数码是一组用来表示某种数制的符号。位权是数码在不同数位上的倍率值。 数制转换 计算机中的二进制 优点 物理上易实现 算术运算规则简单 便于进行逻辑运算 抗干扰能力强 数据存储基本单位 计算机中的存储空间可以看作是一个一个的小格子，一个1或0占用一个格子，称为一位（bit），连续的8位称为一个字节（Byte）。字节是计算机存储数据的基本单位。 数值数据的二进制编码 数值数据需要转换为二进制，以字节为最小单位存储在计算机中。 文本数据的数字化与编码 计算机采用一定的编码方法将文本数据转换为二进制数据。英文字符和中文汉字的编码方法是不一样的。 英文字符的数字化与编码 ASCII码（American Standard Code for Information Interchange，美国信息交换标准码）是当前应用最广泛的一种英文字符编码标准。它使用7位二进制来表示一个字符，存放在一个字节中，最多可以表示2^7=128个字符，包括了10个数字字符、54个大小写英文字母以及英文符号等95个可显示字符，以 及33个控制字符 汉字的数字化与编码 汉字与数值、英文字母一样, 在计算机中也需要采用二进制编码。 汉字编码比英文字符编码要复杂得多。汉字需要用2个字节进行编码。 1980年，中国国家标准总局发布了《信息交换用汉字编码字符集》， 标准号是GB 2312-1980。 GB2312编码适用于汉字处理、汉字通信等系统之间的信息交换。 GB2312编码中规定：每个汉字及符号用两个字节来表示。 第一个字节称为“高位字节”（也称“区字节”） 第二个字节称为“低位字节”（也称“位字节”） Unicode编码 全世界有上百种语言，人们希望有一种编码可以统一表示世界上所有的符号，每个符号在计算机内是唯一的。 为了达到这个目的，国际组织制定了国际统一的字符编码集Unicode。 Unicode是一种可以容纳世界上所有文字和符号的字符编码方案。 Unicode是为了解决传统的字符编码方案的局限而产生的，它为每种语言中的每个字符设定了统一并且唯一的二进制编码，以满足跨语言、跨平台进行文本转换、处理的要求。 Unicode通常用两个字节表示一个字符，Unicode编码分为17组平面，每个平面有216=65536个码位，一共有17×65536=1114112个码位，可以表示一百多万个符号。 多媒体数据的数字化与编码 声音的数字化与编码 采样 声音的采样是指每隔一段时间从连续变化的模拟音频信号中取一个幅度值（也称为采样值），从而把时间上连续信号变成时间上离散信号，即离散化 量化 声音的量化是用二进制数表示采样所得到的幅度值，即将采样得到的离散音频信号转换为有限位数的、规范化的声音数据后，再转换成二进制编码 声音量化时，先将采样后的数据转换为整数，如图中的数据变化幅度从3到15，用四位二进制表示。量化所 用的二进制位数称为量化位数，一般可以设置为8、16、32位等，量化位数越大，声音的幅度分辨率越高， 声音数据占用的存储空间越大，还原后音频的质量越好...
