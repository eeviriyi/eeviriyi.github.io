记录 Blog 的恢复

在续上次计划通过 Github Page 搭建博客后，又出现了页面空白的 Bug，初步排查是 Node.js 的版本问题，这一弄又是两天。中间真的是不知道 Bug 出在哪里一直找一直找，还好最后在逛街的时候想到了思路。

那么目前在服务器上的搭建也是有了一点头绪，不妨采用 服务器 + Github Page 双备份的形式。

发现之前的 Bug 是某些 CDN 被墙的缘故，现在更换了 360 的 CDN

静待花开。

记录建站の无语事件

大学阶段也不太会弄这些东西了，还是不要太麻烦为好。
