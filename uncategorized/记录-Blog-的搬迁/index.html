记录 Blog 的搬迁

个人 Blog 原先部署在 Github Page。众所周知，由于网络管制等诸多原因，Github 在境内访问出奇地慢甚至在有些地方无法访问，而且因为是初次接触 Hexo 这种静态部署，遂决定吧网站搬迁到 GItee Page 上来，并采用的大量的 CDN 加速，希望能有更好的体验。

改动

  • 评论系统由 Valine 转向 Waline
  • 升级 Next 8.X
  • 配置 Lazyload Fancybox 等插件
  • 加密系统 Hexo-blog-encrypt 的引入
  • Front-matter 的学习与规范
  • 使用 SMMS 等图床
  • 本地搜索系统
  • UI 的优化
