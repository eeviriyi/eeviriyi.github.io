变量的赋值及意义

变量是由变量名和变量值两部分组成。中间的 = 是赋值而不是相等的意思，即把右边的变量值输送给左边的变量名。

  • 变量名由字幕（AZ，az）、数字（0~9）、下划线组成，并且第一个字符不能是数字。如果使用非法标识符，运行时会报相关错误。
  • 变量名的字母的大小写是有区分的，程序中同时出现大小写的两个变量，它们是不同的个体。
  • Python中一个变量已经被赋值（有了自己的数据类型），就不能再随便将其作为其它类型来操作

Python 的控制语句

自上而下的顺序

在编写顺序结构算法时，应按照算法中的自上而下的顺序逐步实现。程序自上而下一行一行执行，不能跳过其中某条语句不执行，也不可某条语句执行多次，整个运行过程中没有分支，没有重复，是一种最简单的程序结构。自上而下的结构顺序是Python程序的控制语句中最基本的顺序结构，其中个别语句的位置顺序可以交换，执行时逐句实现。

做出判断的分支

算法进行程序实现时，分支结构可以用if语句来实现。If语句用于实现判断的分支即选择结构。它判断给定的条件是否满足，并根据判断结果决定执行某个分支的程序段。
